# Data Cleaning & Reporting Automation Dashboard

## Overview

This project is developed using Python and Streamlit to automate data cleaning and reporting tasks. It allows users to upload CSV files, clean missing and duplicate values, generate dataset summaries, and visualize numerical data.

## Features

- Upload CSV datasets.
- Remove missing values automatically.
- Detect and remove duplicate records.
- Generate automated data cleaning reports.
- Display dataset statistics and summaries.
- Visualize numeric data using charts.

## Technologies Used

- Python
- Streamlit
- Pandas
- Matplotlib
- OpenPyXL

## How to Run the Project

1. Install the required libraries:


pip install streamlit pandas matplotlib openpyxl


2. Run the project:


python -m streamlit run app.py


## Project Output

- Cleaned Dataset
- Missing Value Report
- Duplicate Value Report
- Dataset Summary
- Visual Data Representation

## Expected Outcome

This project helps users understand data preprocessing, automation techniques, and reporting workflows, improving data analysis efficiency.
